🚀 Começando
Pré-requisitos
.NET 8 SDK

Node.js 18+

Angular CLI 17+

MySQL 8.0+

Instalação
Clone o repositório

bash
git clone https://github.com/salestech-dev/biblioteca-system.git
cd biblioteca-system
Backend (.NET)

bash
cd Backend
dotnet restore
dotnet ef database update
dotnet run
Frontend (Angular)

bash
cd Frontend
npm install
ng serve
